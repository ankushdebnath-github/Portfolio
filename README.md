# Ankush Debnath | Cybersecurity Portfolio

[![Site preview](/public/assets/thumbnail.png)](https://your-live-site-link-here)

This is my personal portfolio website built to showcase my cybersecurity projects, technical skills, certifications, achievements, education, and CV in one place. It is designed as a modern single-page experience with a strong cyber-themed visual style, custom motion, and recruiter-friendly sections.

Built with [Next.js](https://nextjs.org/), [TypeScript](https://www.typescriptlang.org/), [React](https://react.dev/), and [Tailwind CSS](https://tailwindcss.com/).

## Features

- Cyber-inspired portfolio design with animated hero visuals
- Rotating role text and interactive motion effects
- Featured project showcase with custom security-themed visuals
- CV preview and download support
- Dedicated sections for skills, training, certificates, achievements, and education
- Recruiter-focused sections like `Why Hire Me` and `Stats`
- Custom cursor, boot sequence, live alert ticker, and terminal-inspired UI elements
- Responsive layout for desktop and mobile
- Built with the Next.js App Router and optimized for modern deployment

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4

## Development

```bash
git clone <your-github-repo-link>

cd Portfolio-main

npm install

npm run dev
```

Open `http://localhost:3000` in your browser.

You can start editing the portfolio by updating `app/page.tsx` and the components inside `app/components/`.

## Deployment

This portfolio can be deployed easily on [Vercel](https://vercel.com/).

### Local production check

```bash
npm install
npm run build
npm run start
```

### Deploy on Vercel

1. Push this project to GitHub.
2. Go to `https://vercel.com/new`.
3. Import your repository.
4. Vercel will detect Next.js automatically.
5. Click deploy.
